# Compass Visualizer

Use Compass Visualizer to inspect compass tag data and derived orientation
outputs.

## Open a Data File

1. Open Compass Visualizer.
2. Choose a downloaded compass tag data file.
3. Review the loaded time range.
4. Inspect heading, sensor channels, and derived views.
