# Qt Program

Use Qt Program to configure and program supported tags.

## Before You Start

- Install the Ultralight Tags host tools package.
- Connect the base station or programmer.
- Confirm that the target tag is powered and connected.

## Program a Tag

1. Open Qt Program.
2. Select the connected programmer or base station.
3. Choose the firmware image for the tag model.
4. Review the detected tag information.
5. Start programming.
6. Wait for the success message before disconnecting the tag.
