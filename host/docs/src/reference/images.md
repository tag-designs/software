# Screenshots and Images

Store guide images in `docs/src/images/`.

## Naming

Use short, stable names that identify the app and screen:

- `qtprogram-main-window.png`
- `qtcalibrate-collection.png`
- `qtmonitor-live-view.png`
- `btdataviz-plot-view.png`
- `sensorviz-compass-view.png`

## Image Guidelines

- Capture only the relevant application window or panel.
- Use realistic sample data when possible.
- Avoid personal file paths, serial numbers, or private device identifiers.
- Keep image width reasonable for documentation pages.

## Markdown Example

```md
![Qt Program main window](../images/qtprogram-main-window.png)
```
