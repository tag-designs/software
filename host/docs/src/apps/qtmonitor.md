# Tag Monitor

Use Tag Monitor to connect to a tag or base station and inspect live messages.

## Connect to a Device

1. Open Tag Monitor.
2. Select the port or detected device.
3. Start monitoring.
4. Confirm that messages are updating.

## Common Checks

- Device identity and firmware version
- Sensor status
- Battery or power status
- Logging state
- Error messages
